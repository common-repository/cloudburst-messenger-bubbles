Adds a clean, easy-to-use "Messenger Bubble" block to represent chat conversations.
 
== Description ==
 
The "Messenger Bubble" block includes the following features:
* Live preview in Gutenberg editor.
* Fully cuztomizable bubble color.
* Fully customizable font color.
* Optional bubble header for name, timestamp, or other chat notifications.
* Align bubbles left or right to indicate sender/receiver.
* Fit multiple bubbles together using Top, Middle, or Bottom options for a sleek, modern look.
 
== Installation ==
 
1. Upload `cloudburst-messenger-bubbles` folder to your `/wp-content/plugins/` directory.
2. Install and activate the Gutenberg WordPress plugin (if pre WordPress 5.0).
3. Activate the plugin through the 'Plugins' menu in WordPress.
